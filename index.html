Procrastination

Game made for a Game Jam hosted by the RIT Game Dev Club where the theme was "Everyday Tasks." My team decided to make a game where you avoid doing everyday tasks and procrastinate instead. As this was my first none academic game that I liked, I decided continue working on it in my spare time. The node editor is something that was made after the game jam to make things easier for the future.

  • Dialogue System
    • Allows player to talk to different objects
    • Allow the player to make choices
    • Branching dialogue paths
  • Dialogue Node Editor
    • Tool that allows for quicker implementation of dialogue
    • Node based
  • Unity
    • Custom Editors
    • Custom Editor Window
    • Manual Serialization
    • Scriptable Objects
